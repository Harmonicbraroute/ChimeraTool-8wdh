## Feature Modules

### Network Unlock
Remove carrier SIM lock from supported Samsung (all generations S1–S24), Huawei, and select MTK devices. Works on both BYOD and locked-to-carrier variants.

### FRP & Screen Lock Bypass
Factory Reset Protection removal and screen lock bypass (pattern, PIN, fingerprint) without triggering data wipe on supported firmware versions. Covers Android 5–14.

### IMEI Repair & Write
Restore zeroed or corrupted IMEI numbers on Samsung (via EFS partition), Huawei, MTK, and Qualcomm devices. Writes both IMEI 1 and IMEI 2 simultaneously.

### Firmware Flash
Direct firmware loading over USB via:
- Samsung: Odin protocol (AP/BL/CP/CSC/USERDATA)
- MTK: BROM / Preloader mode — scatter file based
- Qualcomm: EDL/9008 mode — QFIL protocol
- Huawei: Hisuite / HiDownload / Emergency

### EFS / NVRAM Backup & Restore
Full partition backup of EFS (Samsung) and NVRAM (MTK) to file. Restore from backup on same or replacement board.

### ADB & Fastboot Tools
Integrated ADB shell, fastboot flash, sideload, unlock bootloader (where OEM unlock is enabled), and partition management — all within the UI, no command line.
